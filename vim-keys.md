# basic

# advanced

## H
Go to top of screen.
## M
Go to middle of screen.
## L
Go to bottom of screen.

## Ctrl + D
Go half page down.
## Ctrl + U
Go half page up.
## Ctrl + F
Go one page down.
## Ctrl + B
Go one page up.

## di"
delete words inside of "
## da"
delete words around "
## @:
repeat the last command
## .
repeat the last operation for editing.
## q:
show commands history
